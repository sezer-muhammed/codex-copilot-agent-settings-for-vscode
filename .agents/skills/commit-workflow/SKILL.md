---
name: commit-workflow
description: Prepare clean, reviewable commit groupings and commit messages.
---

1) Inspect changed files and group related edits.
2) Split unrelated changes into separate commits.
3) Ensure each commit passes relevant checks.
4) Write concise commit messages describing intent and scope.
5) Output:
   - commit sequence
   - per-commit file list
   - per-commit verification notes
